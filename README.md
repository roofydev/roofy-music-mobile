## Features

- **Stream Everything** — Play any song, album, or video from YouTube Music.
- **Background Playback** — Keep the music going while you multitask.
- **Offline Listening** — Download and cache tracks for when you're off the grid.
- **Smart Search** — Find songs, albums, artists, videos, playlists, and podcasts instantly.
- **Live Lyrics** — Sing along with synced lyrics from multiple providers.
- **Library Sync** — Log in to your YouTube Music account to sync favorites, playlists, and history.
- **Personalized Discovery** — Quick Picks tailored to your taste.
- **Social & Stats** — Listen Together, Discord Rich Presence, and Last.fm scrobbling.
- **Fine-Grained Control** — Equalizer, sleep timer, skip silence, audio normalization, tempo/pitch adjustment.
- **Themes** — Light, dark, pure black, and dynamic Material You palettes.
- **Widgets** — Home-screen widgets for quick access to your music.
- **Playlist Power** — Create, import, reorder, and manage playlists with ease.

## Build from Source

Roofy Music is built with modern Android tooling. You'll need **Java 21**.

```powershell
# Windows
.\gradlew.bat :app:assembleFossDebug

# macOS / Linux
./gradlew :app:assembleFossDebug
```

The compiled APK will be available at:
```
app/build/outputs/apk/foss/debug/app-foss-debug.apk
```
